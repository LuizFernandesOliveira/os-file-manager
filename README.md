# github.com/LuizFernandesOliveira/os-file-manager

### pre requisites

- go 1.21
- make
- curl

### Running locally

```shell
make init
```

```shell
make run
```

```shell
curl localhost:8080/public/demos 
```

### Running tests

```shell
make test
```

### Building app

```shell
make build
```

_Check if `.env` file exists._
 
```shell
./application
```
